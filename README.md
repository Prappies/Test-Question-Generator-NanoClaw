## ✨ Features

| Feature | Description |
|---|---|
| 📄 **PDF + URL Support** | Upload a file or paste any webpage link |
| 🧠 **AI-Generated Questions** | Claude reads and creates 20 MCQ questions |
| 📊 **Google Slides Output** | Quiz auto-saved as a presentation in your Drive |
| ✅ **Instant Grading** | Submit answers in Discord, get scored immediately |
| 💬 **Detailed Feedback** | See exactly what you got wrong and why |
| 🔄 **Retry Mode** | Generate new questions based on weak areas |
| 🔐 **Secure OAuth** | Login to Google once — works forever after |
